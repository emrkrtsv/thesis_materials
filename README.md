# thesis_materials
--------------------------------------------------------------------------------------------------------------------------------------------
The materials found during the creation of the thesis are uploaded here.
--------------------------------------------------------------------------------------------------------------------------------------------
The repository contains two versions of the dataset, in an excel format, representing different stages of the annotation process. InclusiveG-EN_1.0 reflects the initial data collected prior to the reconciliation phase. Following this, a second meeting was held with the annotators to address inconsistencies, clarify requirements of the annotation scheme, and rectify errors. As a result, InclusiveG-EN_2.0 was developed as the final, corrected version, incorporating all revisions and final consensus reached during the reconciliation meeting.
--------------------------------------------------------------------------------------------------------------------------------------------
Source code for the calcualtion of IAA and Revealing the Results are provided in two seperate ipynb file. 
--------------------------------------------------------------------------------------------------------------------------------------------
The revised and final version of Annotation Guidelines are provided in a PDF format.
